# 🚀 如何运行和查看前端界面

## 前置要求

确保你的电脑已安装：
- **Node.js** (版本 16 或更高)
  - 下载地址：https://nodejs.org/
  - 安装后，打开命令行输入 `node -v` 检查是否安装成功

## 快速开始

### 1. 安装依赖

在项目根目录打开终端（命令行），运行：

```bash
npm install
```

或者使用 yarn：

```bash
yarn install
```

### 2. 启动开发服务器

```bash
npm run dev
```

或者使用 yarn：

```bash
yarn dev
```

### 3. 查看效果

启动成功后，终端会显示类似信息：

```
  VITE v4.x.x  ready in xxx ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

**浏览器会自动打开** `http://localhost:3000/`，如果没有自动打开，请手动访问这个地址。

## 📱 功能说明

### 桌面端功能
- 顶部导航栏：可以切换不同页面
- 首页：展示系统概况和快捷入口
- 医院列表：查看和筛选医院
- 医生列表：查看医生信息和排行榜
- 在线问诊：与医生聊天
- 预约管理：查看和管理预约
- 就诊档案：查看历史记录和AI分析

### 移动端功能
- 底部导航栏：快速切换主要功能
- 响应式设计：自动适配手机屏幕

## ❓ 常见问题

### 1. 端口被占用
如果 3000 端口被占用，Vite 会自动使用下一个可用端口（如 3001），请查看终端输出的实际地址。

### 2. 依赖安装失败
- 检查网络连接
- 尝试使用国内镜像：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```

### 3. 样式不显示
确保已安装 Tailwind CSS 相关依赖，如果样式异常，尝试：
```bash
npm install -D tailwindcss postcss autoprefixer
```

## 📝 开发提示

- 修改代码后，页面会自动热更新（Hot Module Replacement）
- 所有组件都在 `src/components/` 目录下
- 样式使用 Tailwind CSS，可以直接修改 className
- 数据在 `src/data/mockData.js` 中，可以修改测试数据
